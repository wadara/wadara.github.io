猿でも分かるgit入門

May 9th, 2014

猿でも分かるgit 入門を読みました

rebase -i 使おうと思った

でHEADからHEAD~~までのコミットを一気に編集したりまとめたりできる

edit で編集 squash でまとめる.

こんな風に

コミットコメントに !squash とかいとくと

で

squash! の後ろの文字で前方一致検索して 勝手にsquash してくれる

~と^ の違い

よく分かってなかった

HがHEADだとすると,

  • HEAD^ … G
  • HEAD^2 … C
  • HEAD^2^ … B

@はreflog をさかのぼる

使った事無いけどいつか使うかもしれない
